 Nivelación Python - Sistema de Vehículos

Este proyecto implementa un sistema de vehículos usando Programación Orientada a Objetos en Python.

 Características:
- Clase abstracta (Vehiculo)
- Herencia múltiple (Auto)
- Encapsulamiento con propiedades
- Polimorfismo

 Estructura:
- vehiculo.py → clase abstracta
- gps.py → ubicación
- radio.py → radio
- auto.py → clase principal
- main.py → ejecución

 Ejecución:
run phyton en el main
